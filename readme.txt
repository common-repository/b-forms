== Description ==
This plugin adds your custom built forms in [bforms.app](https://bforms.app) portal to your WordPress projects

= Data Collection =
This plugin is an extension of the application at [bforms.app](https://bforms.app), to bring the forms created at this portal to be available inside WordPress. There is no option to create the forms inside WordPress with this plugin and therefore requires data pulls from [bforms.app](https://bforms.app)

== Frequently Asked Questions ==
= How do i create the form ? =
The Forms have to be created at the official site of B Forms at [bforms.app](https://bforms.app). Creating an account is absolutely free.
= How to add the form in wordpress ? =
Once you create the form in bforms.app, you can copy the token for that form and add it to your wordpress using the shortcode.
= What is the shortcode for adding the forms ? =
There are 2 shortcodes available to use. One is for adding the form and the other to display the submitted values.

1. Adding the Form: [b_form token="abcdefg"] 
*(where abcdefg is the token received from bforms.app)*

2. Displaying the submitted data: [b_form_data token="abcdefg"] 
*(where abcdefg is the token received from bforms.app)*
